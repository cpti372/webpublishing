 html -> 프로그래밍 

웹표준  
 - 디자인, 컨텐츠, 프로그래밍을 분리시킨다. 
 - 디자인 레이아웃은 css를 사용한다.  

 html      -> css    ->    프로그래밍 
(요소          (디자인)     (DB, 로그인, 게시판)
컨텐츠)

>> CSS 디자인이란?
  HTML 요소에 속성과 값을 이용하여 디자인한다. 

>>> CSS 적용방식
1) 인라인 방식 
    <html태그 style="CSS속성1:값1;CSS속성2:값2;">
    HTML 태그안에 style 속성을 이용하여 
    직접 CSS 속성과 값을 지정한다. 
     
    예) h1 태그에 글자색상(color)과 배경색상(background)을 
         지정한다.  
      <h1 style="color:white;background:black">제목</h1>
         
2) head 방식 
   - html 영역 중에서 
     <head> ~</head> 사이에 아래 소스 삽입한다.

    <style type='text/css'>
          셀렉터 {속성1:값1;속성2:값2;}
     </style>
3) 외부 CSS 방식 
   - 별도의 css 파일 생성 
   - 예) style1.css
   - CSS 문서 연결
     head 영역에 아래의 문법으로 삽입
   <link rel="stylesheet" href="css파일 경로">


  셀렉터(선택자) 종류 
    - 태그 선택자
    - 클래스 선택자 
    - 아이디 선택자 
     
// 태그 선택자 :
문서에서 태그선택자로 지정한 
태그가 삽입되면 자동으로 정의한 
디자인이 적용된다. 

<style type='text/css'>
    태그명 {속성1:값1;속성2:값2;}
</style>

// 클래스 선택자 - 도트선택자 
페이지에서 원하는 부분만 적용 가능 

1) 클래스 디자인 정의 
<style type='text/css'>
    .클래스이름 {속성1:값1;속성2:값2;}
</style>

2) 등록한 클래스 디자인을 적용
   <태그명 class="클래스이름">
    ....
   </태그명>


// 아이디 선택자 
페이지에서 원하는 부분만 적용 가능 
문서에서 한번만 적용 가능 

1) 아이디 디자인 정의 
<style type='text/css'>
    #아이디이름 {속성1:값1;속성2:값2;}
</style>

2) 등록한 아이디 선택자 디자인 적용
   <태그명 id="아이디이름">
    ....
   </태그명>

// 트리 선택자 
: 계층 구조 
1) 트리 선택자 정의 1 - 부모요소가 아이디인 경우 
<style type='text/css'>
    #선택자1(아이디) 선택자2 ... {속성1:값1;속성2:값2;}
</style>

2) 트리 선택자 정의 1 - 부모요소가 클래스인 경우 
<style type='text/css'>
    .선택자1(클래스) 선택자2 ... {속성1:값1;속성2:값2;}
</style>    

// 하이퍼링크 관련 선택자 
a:link { 속성1:값1;속성2:값2; } 보통 상태 
a:hover{ 속성1:값1;속성2:값2;  } 가져다대면
a:visited{ 속성1:값1;속성2:값2; } 방문 상태 
a:active{ 속성1:값1;속성2:값2; }  클릭 순간 활성 상태 

css 적용 전상태의 하이퍼링크 스타일   
 <a href="url" target="_blank">글자</a>
보통 - 파란색밑줄 글자 
방문 - 보라색밑줄 글자 
활성 - 빨간색밑줄 글자 

// 그룹 선택자 
: 같은 속성과 값을 가지고 있는 
선택자끼리 쉼표를 이용해서 묶어준다.

//순서 선택자 -
태그명:nth-child(숫자){속성1:값1;속성2:값2; }
태그명:nth-child(odd/even){속성1:값1;속성2:값2; }
 : 홀수번째 짝수번째

태그명:first-child {속성1:값1;속성2:값2; }

태그명:last-child {속성1:값1;속성2:값2; }

// 직계선택자 
선택자>선택자 

// 태그 요소 2가지 스타일 
인라인 요소 - 같은 줄에 오는 형태 
   (이미지,글자, 입력상자 ...)
블록 요소 - 같은 줄에 오지 않는 형태 
   (div, p, table)

//레이아웃 관련 속성 
1) 플로우트 방식 
   float:left/right
     : 부모요소를 기준으로 왼쪽 과 오른쪽으로 배치 
   clear:float 해제. both
2) display 방식 
   none - 보이지않게 지정한다. 
   inline - 블록 요소를 인라인 요소로 변경한다
   block - 인라인 요소를 블록 요소로 변경한다. 
3)position 방식 
   - absolute 
     top(y), left(x) 속성으로 위치 지정 
     기준값이 11시 방향
      z-index:겹치기 순서 
                  : 마지막에 작성한 요소가 위에 배치 
                  : 숫자값이 높을 수록 위에 배치           
   - fixed
      좌표속성 -  bottom(y) /top(y),   left(x)/right(x)
      스크롤바에 상관없이 그자리 유지    

   - relative 
     상대적배치 
     좌표기준이 자기 자신 
     좌표속성 -  bottom(y) /top(y),   left(x)/right(x)         


// padding - 안쪽 여백 
// margin - 바깥 여백 
    margin-left/margin-right
    margin-top/margin-bottom
    margin:숫자px (상하좌우)
    margin:숫자1 숫자2(상하 좌우)
    margin:숫자1 숫자2 숫자3 숫자
       (12시 3시 6시 9시)
   marign에서 좌우 여백을 auto로 
   지정시 중앙정렬된다. 

   margin-left:auto;
   margin-right:auto
    = 
   margin:0 auto
    
// box-sizing :boder-box
- 안에 안쪽여백이나 테두리를 주어도
   가로크기로만 산정 

박스모델 
  - width + border + padding
